# 03 — 장르 분석 (1인 5주 + 웹 + 10분 중독 적합도)

## 평가 척도

각 장르를 다음 5축으로 평가 (1~5):

- **참신성 잠재력**: 트위스트 가능성
- **구현 난이도(역)**: 5주에 1인이 마칠 수 있는가 (높을수록 쉽게 마침)
- **모바일 적합성**: 터치/세로 모드 친화도
- **반복성**: 한 판 30초~2분 반복 구조
- **첫인상 임팩트**: 시연영상·캡처가 매력적인가

## 종합 판단 — 추천 후보 3종

후보를 좁힐 때, "참신성 5점이면서 1인 5주에 가능한" 장르를 우선:

### 후보 1: 리듬 + α (한국어 한글 조합 결합)

> "음악 박자에 맞춰 한글 자모를 조합하는 게임"

- 한글 받침/조합은 다른 나라에 없는 메커닉
- 리듬 게임의 시각·청각 임팩트
- BGM은 무료 라이선스 또는 Web Audio API 자체 생성

### 후보 2: 클리커 + 로그라이트 빌드

> "30초마다 빌드를 새로 짜는 클리커. 한 판 = 30초."

- 짧은 사이클 (10분에 20번 빌드)
- 매 판 새 무작위성 (변동 보상)
- 시각적으로 단순해도 OK (BGM·이펙트로 보강)

### 후보 3: Suika + 디펜스 (혼합 장르)

> "위에서 떨어지는 머지 결과물이 아래 적을 막는 타워가 된다"

- Matter.js로 빠른 구현
- 두 장르 결합 자체가 트위스트
- 모바일 친화 세로 화면
