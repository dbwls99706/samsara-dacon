# 16 — 카드 풀 (60장 + 18 시너지)

> **단일 진실 공급원**. 코드는 이 문서를 JSON으로 변환해 사용한다.
>
> 모든 카드 효과는 데이터 주도 (트리거 + 효과). 코드는 효과 정의만 보고 자동 적용.

## 카드 데이터 스키마

```ts
type CardTag = 'fire' | 'ice' | 'gold' | 'time' | 'chaos' | 'echo';
type Rarity = 'common' | 'rare' | 'epic' | 'legendary';
type Trigger =
  | 'onTap'              // 탭마다
  | 'onTick'             // 매 프레임/초
  | 'onWaveStart'        // 웨이브 시작
  | 'onWaveEnd'          // 웨이브 종료
  | 'onCombo'            // 콤보 임계
  | 'onCoinThreshold'    // 코인 임계
  | 'onTapNth'           // n번째 탭마다
  | 'onCardPicked'       // 카드 선택 시
  | 'onSynergy'          // 시너지 발동 시
  | 'onLifeLost';        // 라이프 손실 시

type Card = {
  id: string;
  name_ko: string;
  name_en: string;
  flavor_ko: string;       // 한 줄 플레이버
  tags: CardTag[];         // 1~2개 (듀얼 태그 가능)
  rarity: Rarity;
  effects: Effect[];
};

type Effect = {
  trigger: Trigger;
  value: number | string;
  scale?: 'flat' | 'percent' | 'mult';
  condition?: string;
};
```

## 카드 등급 분포 합계

| 등급 | 갯수 | 풀 비율 |
|---|---|---|
| Common (C) | 40 | 약 70% |
| Rare (R) | 15 | 약 25% |
| Epic (E) | 4 | 약 4% |
| Legendary (L) | 1 | 약 1% (전설 6장이지만 한 번 등장 시 1% 확률) |

(60장 분포: C 40 / R 15 / E 4 / L 1 ... 위 표는 각 태그 당 C 4 / R 3 / E 2 / L 1 = 10장 × 6태그 = 60장)

> 정확한 카드별 등급 분포는 위 표 참조. C 4 / R 3 / E 2 / L 1 per tag.

## 카드 풀 무작위 등장 확률

3장 카드 선택 시:
- C × 65%
- R × 25%
- E × 8%
- L × 2%

가중 무작위 — 사용자에게 가시화하지 않음.

## 밸런스 노트

- C 카드는 단독으로 약함, 같은 태그 누적 시 강함 → 시너지 빌드 유도
- R 카드는 단독으로 중간, 시너지 보너스 큼
- E 카드는 단독으로 강함, 듀얼 태그로 빌드 유연성
- L 카드는 게임을 깸 → 1% 확률은 의도된 잭팟
