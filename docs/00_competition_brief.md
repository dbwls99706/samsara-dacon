## 평가 (2단계)

### 1차 — 대중 투표 (상위 10팀 선정)

**가중치 합산식**:
```
점수 = 제출팀 투표 × 0.6 + 참가팀 투표 × 0.2 + 대중 투표 × 0.2
```

- 제출팀 = 산출물을 제출한 팀들의 투표
- 참가팀 = 신청만 한 팀까지 포함하는 투표
- 대중 = 외부 일반 투표

> **시사점**: 다른 제출팀(=동료 개발자)이 인상받을 수 있어야 한다. 기술적 디테일·완성도·디자인 감각이 보이는 게임이 유리.

**동점자 처리**: 참가팀 투표 우선 → 산출물 최초 업로드 시간이 빠른 팀 우선.
> 즉, **산출물을 일찍 올릴수록 동률에서 유리**하다. 6/8 마감 직전 제출보다 6/4~6/5 제출이 동점 깨기에 좋다. (단, 완성도 25점이 1순위니까 무리하게 일찍 내는 건 X.)
