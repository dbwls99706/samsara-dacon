# 17 — 도파민 시스템 상세 사양

> 7 도파민 레이어의 구체 사양. 카드는 `docs/16_card_pool.md`. 본 문서는 카드 외 모든 시스템.

## 1. 탭 피드백 폭격 (Layer 1)

### 매 탭마다 동시 발생 (8가지)

```
0ms      pointerdown 이벤트
1ms      이벤트 → 액션 변환 (동기)
2ms      hitstop 시작 (50ms — 게임 시간 정지, 입력은 받음)
2ms      화면 ripple (탭 좌표 중심 0.4초 원형 확산)
2ms      캐릭터 squash (Y축 0.85 → 1.0 / 200ms ease-out)
2ms      색 플래시 (백그라운드 밝기 +10% / 100ms)
2ms      파티클 5~8개 (방사형, 1초 fade)
3ms      숫자 팝업 (탭 가치 + 호 그리며 위로 600ms / fade-out)
3ms      SFX 레이어드 (베이스 톤 + 탑 톤 / 80ms)
4ms      화면 우측 데미지 로그 (1줄, 4초 후 사라짐)
50ms     hitstop 종료, 게임 시간 정상화
```

### 파라미터 튜닝

| 파라미터 | 기본 | 콤보 5+ | 콤보 25+ |
|---|---|---|---|
| Hitstop | 50ms | 70ms | 100ms |
| 파티클 갯수 | 5 | 10 | 25 |
| 파티클 색 | 흰색 | 노랑 | 황금/다채 |
| Squash 강도 | 0.15 | 0.25 | 0.40 |
| 숫자 팝업 크기 | 24px | 32px | 48px |
| Ripple 알파 | 0.3 | 0.5 | 0.9 |

## 2. 콤보 캐스케이드 (Layer 2)

### 콤보 윈도우

- 기본: **0.3초** 이내 다음 탭 시 카운트 증가
- 카드/시너지로 확장 (최대 0.6초까지)
- 윈도우 만료 = 콤보 0으로 리셋

### 콤보 임계값 효과

| 콤보 | 효과 | 시각 | 청각 |
|---|---|---|---|
| ×3 | 점수 ×1.1 | 작은 스파크 | 톤 ↑ |
| ×5 | 점수 ×1.2 | 흰 스파크 | 청량 톤 |
| ×10 | 점수 ×1.5 | 컬러 오라 + 화면 가장자리 발광 | 화음 (3음) |
| ×25 | 점수 ×2 + **시간 정지 0.5초 + 폭발** | 풀스크린 흰 플래시 | 베이스 드럼 |
| ×50 | 점수 ×3 + **FRENZY 모드 5초** (게임 1.5배속) | 화면 흔들림 + 빨강 톤 | BGM 추가 트랙 |
| ×100 | 점수 ×5 + **GOD MODE 5초** (전체 ×10) | 황금 폭우 + 화면 풀 황금 | 코러스 |
| ×200 | 점수 ×10 + 영구 트로피 | 풀스크린 시네마틱 0.5초 | 풀 오케스트라 |
| ×500 | 점수 ×100 + 비밀 업적 | 시네마틱 1초 | 풀 사운드 |

### 콤보 끊김 처리

- 시각: 화면 살짝 회색화 0.2초
- 청각: 짧은 "fft" 톤
- 알림: 화면 중앙 "콤보 ×N 끊김" 1초 표시
- **카드 I04 / E04 / I01 보유 시**: 끊겨도 일부 유지

## 4. 보스 / 의식 시스템 (Layer 5)

### 보스 종류

| 종류 | 등장 | 목표 | 보상 |
|---|---|---|---|
| 일반 보스 | W5, 15, 30, 40 | 30초 안에 목표 점수 격파 | 의식 — 영구 런 버프 |
| 메가 보스 | W10, 20, 35 | 30초 안에 더 높은 목표 | 의식 + 메타 보상 (스킨/BGM 해금) |
| 신성 보스 | W25, 50, 75 | 까다로운 목표 (예: 콤보 50 유지) | 의식 + 전설 카드 직접 선택 |

### 의식 화면 (보스 격파 후)

```
보스 격파 시간: 18.4초 (보너스 +50%)
   ───────────────────────
의식의 카드 3장 (영구 런 버프):

  [1] 격노      — 보스 격파 시 +5,000
  [2] 영원의 약속 — 라이프 영구 +1
  [3] 시간 정복  — 모든 보스 시간 -5초
```

3장 중 1장 선택. 영구 (이번 런 한정).

### 보스 시각/청각

- 등장 시: 화면 어두워짐 + 보스 등장 컷 (1초)
- BGM 변경 (보스 전용 강한 톤)
- 격파 시: 풀스크린 폭발 + 0.5초 정지 + "VICTORY" 표시
- 격파 보너스: 격파 시간이 빠를수록 점수 ×% 추가

### 보스 목표 점수 공식

```
target = 100 * 1.4^wave
보스 웨이브: target *= 2.5
메가 보스: target *= 4
신성 보스: target *= 7
```

## 6. 메타 진보 (Layer 6) — 윤회 시스템

### 환생 포인트(RP) 계산

```
RP = floor(누적코인 / 10000)
```

게임 오버 시 누적 코인을 RP로 환산. 메인 화면에서 RP 표시.

### 메타 상점 메뉴

```
─── 환생의 사원 ───

🔓 잠금 해제 (1회 구매)
  - 카드 1장 (RP 5)            ▶ 진행 중 [12/30]
  - 모디파이어 1종 (RP 8)      ▶ [3/15]
  - 캐릭터 스킨 (RP 30)         ▶ [1/5]
  - BGM 트랙 (RP 50)            ▶ [1/4]

📈 영구 강화 (반복 가능)
  - 시작 라이프 +1 (RP 100, 3회까지)
  - 시작 카드 슬롯 +1 (RP 200, 1회)
  - 시작 코인 +500 (RP 50, 5회까지)
  - RP 환산 효율 +10% (RP 75, 3회까지)
```

### 메인 화면 진화

| 누적 사이클 | 변화 |
|---|---|
| 0 | 검은 배경, 캐릭터 작게 |
| 100 | 별 등장 |
| 250 | 산 실루엣 |
| 500 | 새 캐릭터 등장 (동행) |
| 1000 | 노을 색감 |

## 7. 일일 시드 + 리더보드 (Layer 7)

### 일일 시드 시스템

```ts
function dailySeed(date: Date): number {
  const ymd = `${date.getFullYear()}${date.getMonth()+1}${date.getDate()}`;
  return hashStringToInt(ymd) >>> 0;
}
```

- 매일 자정 KST: 모든 플레이어 동일 시드
- 시드로 결정되는 것:
  - 시작 카드 3장 (메인 화면에서 미리 보여 흥미 유발)
  - 첫 5웨이브 모디파이어
  - 첫 2 보스 (정확한 목표 점수)
- 일일 점수는 별도 통계

### 리더보드 (Supabase)

#### 테이블 스키마

```sql
create table leaderboard (
  id bigint primary key generated by default as identity,
  date_seed text not null,        -- "20260504" 형식
  nickname text not null check (length(nickname) <= 16),
  score bigint not null,
  run_identity text,
  card_ids text[],                -- 빌드 공유용
  created_at timestamptz default now()
);

create index idx_leaderboard_date_score on leaderboard(date_seed, score desc);

alter table leaderboard enable row level security;

-- 누구나 조회 가능
create policy "leaderboard_read"
  on leaderboard for select
  using (true);

-- 익명 사용자도 INSERT 가능
create policy "leaderboard_insert"
  on leaderboard for insert
  with check (
    length(nickname) between 1 and 16
    and score > 0
    and score < 1e18  -- 부정 점수 방지
  );

-- UPDATE/DELETE 차단
```

#### 비속어 필터

- 한국어 욕설 사전 (오픈소스 또는 자체)
- 정규식 우회 패턴 차단 ("ㅅㅂ", "ㅄ" 등)
- 검출 시 자동 마스킹 ("***")

#### 일일 1회 갱신

- localStorage에 마지막 제출 시드 + 점수
- 같은 시드에 더 높은 점수 시만 새 INSERT
- 부정 방지: 클라이언트 측 IP rate limit (Supabase 함수)

### 메인 화면 리더보드 위젯

```
─── 오늘의 챔피언 ───
1. 김XX     1.4e9   🔥🔥🔥 불의 황제
2. 이XX     8.2e8   💰💰💰 황금 폭군
3. 박XX     5.1e8   🪞🪞🪞 거울 마법사
...
[내 점수 보기]   [어제 시드 보기]
```

### 공유 이미지 자동 생성

- Canvas로 PNG 생성 (1080×1080 정사각)
- 점수, 닉, Run Identity, 카드 5장
- "공유" 버튼 → Web Share API → 카톡/X/인스타
- 폴백: 다운로드

## 9. 핫스팟 시스템 (액티브 스킬)

### 사양

- 매초 화면에 1개 황금 영역 등장
- 크기: 80px × 80px
- 위치: 화면 무작위 (가장자리 50px 마진 안)
- 지속: 1.0초 (페이드 인 0.2 / 유지 0.6 / 페이드 아웃 0.2)
- 그 위 탭 = ×3 점수
- 시각: 황금 발광 + 회전

### 변종

- 게임 진행에 따라:
  - W3+ → 핫스팟 등장 빈도 ↑
  - W6+ → 동시 2개
  - W10+ → 동시 3개 + 일부는 빨강 (×0.5 트랩)
  - W15+ → 짧은 지속 (0.5초)

## 11. 하이라이트 릴 (게임 오버 마무리)

### 자동 편집 알고리즘

게임 진행 중 다음 이벤트 캡처:
- 가장 큰 콤보 도달 순간
- 가장 비싼 시너지 발동
- 보스 격파 순간
- 전설 카드 획득
- 가장 높은 단일 점수 획득

게임 오버 시 5초 자동 편집 (ffmpeg 없이 클라이언트 측 합성):
- 각 이벤트 1초씩 (5개)
- BGM 깔린 상태
- 자막 (이벤트 명칭)
- 다운로드 가능 (선택)

### 시각 캡처
