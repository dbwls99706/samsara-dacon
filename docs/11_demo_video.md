# 11 — 시연 영상 (YouTube) 스크립트

> 시연 영상은 1차 대중 투표 + 2차 심사위원 평가 모두에 영향을 준다. **게임 다음으로 중요한 산출물.**

## 1. 사양

- **플랫폼**: YouTube (공개 또는 일부공개)
- **길이**: **60~90초** (절대 5분 넘기지 말 것)
- **해상도**: 1920×1080 (16:9), 60fps 권장
- **자막**: 한국어 + 영어 동시 (또는 한국어 + 영어 자막 토글)
- **음성**: 나레이션 X, BGM + 효과음만 (가능)

## 2. 필수 포함 항목 (DACON 명시)

✅ 시작 화면
✅ 주요 플레이 구간
✅ 핵심 시스템 (점수, 콤보, 강화 등)
✅ 종료 화면

> "데모 영상만으로 게임 전체 단계를 확인할 수 없는 경우" = **실격**.

## 3. ⭐ 정밀 촬영 콘티 (75초, 1등 최적화) — 이대로 찍어라

> **전략**: 우리의 약한 평가축은 **참신성(20)** 과 **재미(15)** 다(완성도/UI/기획은 이미 강함).
> 그래서 **첫 8초에 "윤회=매판 다른 운명" 참신성 훅**을 박고, **타격감(콤보·궁극·시너지 폭발)을
> 전면**에 내세운다. 5개 평가축을 의도적으로 전부 건드린다. 평균 컷 2~3초, 결정적 순간만 슬로우.
>
> ⚠ 규정: 시작/주요플레이/핵심시스템/종료 **전 단계 미포함 = 실격.** 아래 콘티는 전부 포함한다.

```
[HOOK — 참신성 0:00–0:08] ★ 가장 중요. 여기서 안 잡으면 투표·심사 다 진다.
0:00–0:03  로고 X. 곧장 빠른 몽타주 — 같은 호랑이 캐릭터, 3개의 완전히 다른 런이
           0.8초씩 점프컷(불꽃 빌드 / 얼음 빌드 / 황금 빌드 — 화면 색·이펙트가 확 다름).
           자막(큰 글씨): "같은 시작, 매번 다른 운명"
0:03–0:08  로고 "SAMSARA · 윤회" + 카피 "30초마다 새 운명을 짠다" 임팩트 등장 + BGM 드롭.

[코어 루프 — 사용성/재미 0:08–0:24]
0:08–0:12  메인 화면(깔끔) → "▶ 시작" 탭. (첫 방문 단순 메뉴 = 직관성 어필)
0:12–0:16  플레이 시작. 하단 힌트 "드래그로 이동·무기는 자동 공격" 보이고 → 적 처치,
           코인/젬 흡수, 콤보 숫자 ×3 → ×10 상승(타격감 SFX 부각).
0:16–0:24  XP 레벨업 → **카드 선택 3장 슬로우 0.5s**(빌드 선택 = 깊이 시그널) → 1장 픽 →
           무기 강화 즉시 반영. 자막: "30초마다 카드 선택 — 빌드를 짠다"

[핵심 시스템·깊이 — 참신성/기획 0:24–0:46] ★ 동료 개발자 표심은 여기서 갈린다.
0:24–0:30  같은 태그 누적 → **시너지 발동 연출**(3장 → 5장 진화 배너 "불의 심장").
           화면 셰이크 + 파티클. 자막: "같은 태그 5장 = 진화"
0:30–0:38  **7장 궁극 발동** = 영상의 클라이맥스. 0.5x 슬로우 + 줌인. 거대 폭발/코인 폭우.
           Run Identity 배너 "★ 태양의 화신 — 콤보 보너스 ×2" 뜨는 순간 포착.
0:38–0:46  보스 등장(빨강 위협 연출) → 격파 → **의식(ritual) 카드 선택** 황금 연출.
           자막: "보스 격파 → 의식으로 운명 강화"

[윤회 메타 — 차별화 결정타 0:46–1:02] ★ 우리만의 참신성. 반드시 명시적으로.
0:46–0:52  게임 오버("사망") → 점수 정산 + "+RP" 획득 연출.
0:52–1:02  **다음 런 시작 → 이전 생의 카드가 자동 계승되는 순간** 클로즈업.
           자막(강조): "죽음은 끝이 아니다 — 이전 생의 힘이 다음 생으로 계승된다 (輪廻)"
           이어서 메인 화면이 사이클마다 진화하는 컷 + RP 상점 영구 잠금해제 1컷.

[마무리 — 완성도/반복성 + CTA 1:02–1:15]
1:02–1:08  데스크톱 화면 + 우하단 **모바일 PIP**(반응형 = 완성도). 일일 시드/리더보드 1컷.
1:08–1:12  "한 판 더" 빠른 점프컷 2개(매번 다른 빌드) — 반복 플레이 욕구 시각화.
1:12–1:15  로고 + 🌐 배포URL + 💻 GitHub + (QR). "키·설치·로그인 0건 — 바로 플레이"
```

### 콘티 촬영 노트
- **0:00–0:08 훅이 전부다.** 투표자는 8초 안에 계속 볼지 정한다. 몽타주 3컷은 색·이펙트가
  확 다른 빌드를 골라(불=핑크/얼음=시안/황금=골드) "매번 다르다"가 한눈에 보이게.
- **궁극(0:30)과 윤회 계승(0:52)** 두 컷이 최고 임팩트 — 여기만 슬로우+줌. 나머지는 빠른 컷.
- 점수는 **0에서 시작**(메타 초기화 후 녹화)해 성장 곡선이 보이게. 단, 궁극/시너지 보여주려면
  같은 태그를 몰아 픽해야 하니 **5~10분 풀 녹화 후 좋은 런만 컷**.
- 자막은 위 문구 그대로(짧고 강하게). 한국어 큰 글씨 + 영문 작게 보조.

## 5. 자막 가이드

- 한국어 자막은 큰 글씨 (화면 1/8 크기)
- 영문 자막은 작게 보조 (필요 시)
- 자막은 1줄 max 12자 (한국어), 24자 (영문)
- 자막 출력 시 게임 화면 가리지 않도록 하단 1/5 영역

## 8. 컷 편집 원칙

- **평균 컷 길이 2~4초** (긴 컷 X — 시청자 이탈)
- **전환은 컷 (Cut)** 만 사용 (페이드 자제 — 시간 낭비)
- 결정적 순간엔 **0.5x 슬로우 모션** + 줌인 (impactful)
- 음악은 1곡 풀 사용 (CC-BY 또는 자체 생성)
- 효과음 강조 (점수, 콤보 시 SFX 부각)

## 9. 음악 (저작권 안전)
