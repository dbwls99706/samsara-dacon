## 1. 게임 개요

### 1-1. 한 줄 정의

> **30초마다 끝나고, 다시 태어나는 클리커. 매 죽음이 새로운 빌드, 매 빌드가 새로운 운명.**

### 1-2. 컨셉

#### 한 줄 정의

**SAMSARA(윤회 / 輪廻)** 는 *30초 사이클 카드 빌드 + 서바이벌 + 게임이론 맵 결정* 의 세 축이 한 화면에서 동시에 돌아가는 웹 미니게임이다.

#### 30초 사이클 카드 빌드

- 한 사이클(웨이브)이 30~35초 만에 끝나고, 매번 무작위 카드 3장 중 1장을 선택해 다음 사이클에 영구 적용한다.
- 카드 60장은 6 태그(🔥불·❄️얼음·💰금·⏱️시간·🌀카오스·🪞에코)로 분류되며, 누적 조합이 28종 **빌드 정체성(Run Identity)** 중 하나를 발현시킨다.
- 같은 태그 3·5·7장이 모이면 시너지가 활성화되어 점수 곡선이 폭발한다 (자세한 풀은 §4-1.5, §4-1.6).

#### 게임이론으로 설계된 맵 오브젝트 (핵심 차별점)

맵 위 13종 오브젝트는 *장식*도 *자판기 보상*도 아니다. 각각이 명확한 게임이론 원리에 입각해 비자명한 선택을 강제한다.

| 오브젝트 | 결정 트리 | 적용 원리 |
|---|---|---|
| 사당(shrine) | 즉시 파괴해 코인 ↔ 60px 내 3초 정지해 영구 +1 maxHP (상호 배타) | Stag Hunt |
| 등불(lantern) | 80px 안에 머물면 무기 쿨다운 -30% ↔ 적이 30px 내 2초 점거 시 버프 박탈 | Tragedy of Commons |
| 저주 토템(cursed_totem) | 파괴 시 거대 보상 (gem×3 + chest×2 + 코인 폭우) **AND** 엘리트 적 3마리 즉시 스폰 | 극단 Stag Hunt (완전 commitment) |

13종 전체 사양은 §5 참조. 적용 원리는 Nash equilibrium · Stag Hunt · Tragedy of Commons · Chicken Game · Information Asymmetry 등 7개.

#### 절차적 지형 (4 biome)

펄린 노이즈 기반으로 생성된 4 biome(산악·평원·저주·성역) 영역에 prop 들이 자연 클러스터링 된다. 산악엔 바위·모놀리스가, 저주 영역엔 cursed_totem·압전판이 우세하게 등장 — 같은 시드라도 어느 biome 에 스폰됐는지에 따라 동선이 달라진다.

#### 반복 플레이 곡선

- 10분 한 세션 = 약 18~25 사이클. 카드 풀이 무작위이므로 **같은 게임이 다시 나오지 않는다**.
- 점수는 1에서 시작해 1조(10¹²)를 넘어 지수적으로 폭발한다.
- 콤보·시너지·보스 격파·맵 결정의 누적 도파민 곡선이 평가 항목 *"10분 안에 핵심 재미 전달, 반복 플레이 유도"* 에 대한 직접적 답이다.

### 1-2.5. 세계관 — 사후 우주 전쟁

> 죽은 자들은 사라지지 않는다. 별이 된다.

#### 설정

플레이어는 **죽음의 순간 동물(호랑이·까치·도깨비·구미호·용)로 환생한 영혼**이다.

환생한 곳은 인간계가 아닌 **사후 우주 — 별과 별 사이의 윤회의 전장**. 이곳에서는 사령신(Death Lord)이 별들을 잡아먹어 영혼들을 영원의 소멸로 끌고 간다.

적은 4종:

- **잡귀** — 운석 조각
- **원귀** — 흩어진 별자리 영혼
- **외계 마귀** — 사이버 도깨비
- **사령 호랑이** — 장산범

#### 카드 = 별의 인장

플레이어가 가진 카드는 **별의 인장**이다. 6개 별자리 태그를 조합해 자신만의 우주적 정체성을 발현시킨다.

| 글리프 | 한글명 | 영문명 |
|---|---|---|
| 🔥 | 화염성 | Fire |
| ❄️ | 빙결성 | Ice |
| 💰 | 황금성 | Gold |
| ⏱️ | 시간성 | Time |
| 🌀 | 혼돈성 | Chaos |
| 🪞 | 반향성 | Echo |

같은 태그 7장이 모이면 별자리가 완성되어 궁극기(7-tier 시너지)가 깨어난다.

#### 비주얼 톤

- **컬러**: 한국 신화의 단청(빨강·금·시안) + 네오펑크 우주.
- **적 디자인**: 보랏빛 운석 / 별자리 영혼 / 사이버 도깨비.
- **맵 데코**: 차원 게이트, 영혼 등불(魂), 봉인 사당(封), 난파한 신단 잔해, 부유 운석.
- **배경**: 4색 성운(보라·시안·핑크·황금) + 250개 별이 패럴렉스로 깜빡임.

한 마디로 **"한국의 사령전(死靈戰)을 우주 스케일로 옮긴 게임"**.

### 1-2.6. 정체성 진술 (Identity Statement)

> 산재한 컨셉·세계관·기술 사양을 **하나의 핵심 정체성**으로 응축한다. 본 진술은 향후 모든 디자인 결정의 기준선이다.

#### 한 문장 캡슐 (Capsule)

> **SAMSARA 는 *30초마다 죽고 다시 태어나는 30초 사이클*에 *60장 카드 빌드*와 *13종 게임이론 prop 결정 트리*를 얹어, 매 죽음이 새 운명이 되는 한국적 윤회 도파민 클리커다.**

이 한 문장에 다섯 축이 모두 들어 있다.

1. **핵심 루프** — 30초 강제 사이클
2. **깊이 축 ①** — 60카드 → 28 Run Identity
3. **깊이 축 ②** — 13 prop 게임이론 결정
4. **메타 서사** — 윤회 / 한국
5. **정서** — 도파민 클리커

마케팅 카피 *"30초마다 새 운명을 짠다"* 는 이 문장의 정서 압축본이다.

#### 5 설계 원칙 (Design Pillars)

본 게임이 무엇이고 무엇이 아닌지를 결정하는 5개 축. 모든 기능은 이 5축 중 ≥2개에 기여할 때만 채택한다.

| # | 원칙 | 구체화 | 위반 시 거부 |
|---|---|---|---|
| **P1** | **30초가 신성하다** | 한 사이클은 30~35초로 고정. 더 길게 만들고 싶은 욕망이 들면 *카드 수* 가 아니라 *결정의 밀도* 를 늘린다. | 60초 보스, 분 단위 컷씬, 긴 튜토리얼 |
| **P2** | **모든 결정은 trade-off 다** | "그냥 좋은 카드/prop" 0개. 모든 카드·prop·모디파이어가 (얻는 것 ↔ 잃는 것) 또는 (즉발 ↔ 누적 commitment) 의 형태. | 무조건 +X% 보상 카드, 자판기 보상 상자 |
| **P3** | **죽음은 진보다** | 게임 오버는 끝이 아니라 RP 환산 + 카드/스킨/BGM 잠금해제. 매 죽음마다 다음 윤회가 조금 더 강해진다. | 패배 시 진보 0건, 그라인드 강제 |
| **P4** | **읽지 않아도 알 수 있어야 한다** | 텍스트 튜토리얼 0건. 첫 판 자체가 학습. 13 prop 은 머리 위 이모지·HP 바·컬러 코드(★황금/⚠빨강/❄시안)로 한눈에 의도 인지. | "시작 전에 읽으세요" 화면, 모달 도움말 |
| **P5** | **검증 가능해야 한다** | 모든 주장은 회귀 테스트(192 spec / 14 files) + Playwright E2E + fresh 빌드로 1:1 검증. README 의 수치는 항상 직전 빌드 실측. | 과장된 수치, "곧 추가 예정" 기능 |

#### 우리가 아닌 것 (What We Are Not)

정체성은 *무엇이 아닌가* 로 더 선명해진다. SAMSARA 는 다음이 **아니다** :

- ❌ **Cookie Clicker 류 무한 클리커** — 30초마다 강제 컷. 무한이 아니라 *반복되는 끝*.
- ❌ **Slay the Spire 류 1시간 덱빌더** — 한 빌드는 1~3 세션. 한 세션은 9~12.5분.
- ❌ **Vampire Survivors 류 자동 진행** — 능동 탭 + 능동 카드 선택 + 능동 prop 결정. 자동은 무기 발사뿐.
- ❌ **장식 맵 + 자판기 보상 상자** — 13 prop 모두 게임이론 결정 트리. 통과만 하면 아무것도 안 줌.
- ❌ **회원가입/결제/광고 게임** — 외부 키 0건, 심사자가 URL 한 번에 즉시 플레이.
- ❌ **번역기 돌린 한국 게임** — 윤회·태그명·캐릭터·prop 디자인 모두 한국 신화/철학에서 출발 (수출 가능한 한국적 정체성).

#### 정체성 → 평가 5축 직접 매핑

본 정체성 5원칙이 100점 룩업 테이블에 어떻게 직결되는지:

| 원칙 | 평가 항목 (배점) | 어떻게 점수가 되는가 |
|---|---|---|
| P1 30초가 신성하다 | 재미·몰입 (15) + 사용성 (20) | 짧은 사이클 = 학습 부담 ↓ + 반복 플레이 트리거 |
| P2 모든 결정은 trade-off | 참신성 (20) + 일관성 (20) | 게임이론 13 prop 매핑이 이 원칙의 기술적 증명 |
| P3 죽음은 진보다 | 재미·몰입 (15) | 메타 진보 RP / 윤회 도감 / 사이클별 메인 화면 진화 |
| P4 읽지 않아도 알 수 있다 | 사용성/UI/UX (20) | 0 텍스트 튜토리얼 + 이모지/HP 바/컬러 인지 + 색약 모드 |
| P5 검증 가능해야 한다 | 완성도/안정성 (25) | **192/192 vitest + Playwright E2E + tsc strict + 워닝 0** |

> **5축 100점 중 100점이 이 정체성 5원칙으로 회수된다.** 즉, 본 정체성은 평가 항목을 점수로 환산하기 위한 *역설계* 의 결과물이기도 하다.

### 1-3. 타깃

- **1차**: 점심·지하철 5~10분의 짧은 휴식을 채우는 모바일 캐주얼 플레이어
- **2차**: 빌드 다양성을 즐기는 로그라이트·클리커 코어 게이머 (Vampire Survivors, Balatro 팬)
- **플레이타임**: 한 사이클 30~35초, **한 세션 9~12.5분 (18~25 사이클)**, 한 빌드 1~3 세션. 첫 게임 오버까진 평균 7~10분.

### 1-4. 플랫폼

- 웹 브라우저 (모바일 + 데스크톱 동시 지원)
- 별도 설치/회원가입/결제 없음
- 한국어 + 영어 동시 지원

## 2. 핵심 규칙

### 2-1. 승리 / 패배 조건

- **승리 조건**: 명시적 승리 없음 (무한 진행). 다만 점수 1e15 도달 시 **초월 엔딩** 발동, 이후 영구 트로피 등록.
- **패배 조건**: 라이프 0. 라이프는 시작 3개. 보스 웨이브에서 목표 점수 미달 시 라이프 -1.

### 2-2. 점수 산정

```
사이클 점수 = Σ(탭 가치) + Σ(자동 누적) + Σ(이벤트 보너스)

탭 가치 = 베이스 × 카드 효과 × 콤보 배수 × 모디파이어 × 시너지 보너스 × 핫스팟 보너스
```

콤보 배수 (×3 → ×500), 시너지 (3장 +30% / 5장 명명 능력 / 7장 게임 깨짐), 모디파이어 30종, 보스 격파 시간 보너스 등이 곱셈으로 누적된다.

### 2-3. 진행 시스템

- **로컬 최고 점수** — localStorage 저장
- **누적 코인 → 환생 포인트(RP)** — 메타 통화 (1만 코인 = 1 RP)
- **환생의 사원 (메타 상점)** — RP 로 영구 강화. 가격표:

  | 카테고리 | 항목 | 가격 | 최대 단계 |
  |---|---|---|---|
  | 잠금 해제 | 카드 1장 | 5 RP | 60장 |
  | 잠금 해제 | 모디파이어 1종 | 8 RP | 30종 |
  | 잠금 해제 | 캐릭터 스킨 1종 | 30 RP | 5종 (호랑이/까치/도깨비/구미호/용) |
  | 잠금 해제 | BGM 변주 1개 | 50 RP | 4 레이어 |
  | 강화 (큰) | 시작 라이프 +1 | 100 RP | 3 단계 |
  | 강화 (큰) | 시작 카드 슬롯 +1 | 200 RP | 1 단계 |
  | 강화 (큰) | 시작 코인 +500 | 5 RP | 5 단계 |

### 4-6. 맵 오브젝트의 결정 트리

기존 서바이벌 장르의 맵 데코는 대부분 장식이거나 자판기형 보상 상자였다. SAMSARA 는 13종 맵 오브젝트(prop)를 모두 **게임이론에 입각한 의사결정 노드**로 설계했다. 자세한 매핑은 §5 참조. 핵심 임팩트는 다음과 같다:

- **이동 의사결정의 부활** — 단순 "적 피하기" 가 아니라 "어느 오브젝트에 commitment 할 것인가" 로 격상.
- **빌드와 직교하는 깊이** — 카드/시너지가 화력 빌드를 만든다면, 맵 결정은 위치 게임 빌드를 만든다.
- **즉발성 vs 누적성** — 13종 중 7종은 즉발 보상, 6종은 위치/시간 commitment 가 필요한 누적 보상.
- **지형 자체가 결정** — 펄린 노이즈 elevation 으로 RIDGE/PEAK 셀에 rocks/monolith 가 산맥 형태로 클러스터링. 오르막은 이동속도 −30%, 내리막은 +25% (slope speed mul) — 적 추적/회피 라인 선택이 elevation 그라데와 직결.

## 5. 맵 오브젝트의 게임이론

### 5-1. 설계 철학 — "모든 오브젝트는 결정이다"

게임 디자인 이론에서 가장 자주 인용되는 명제는 시드 마이어(Sid Meier)의 *"게임은 흥미로운 결정의 연속이다"* 다. SAMSARA 는 이 명제를 맵 레벨까지 확장했다. 13종 prop 모두 게임이론적 의사결정 트리를 가지며, 다음 7개 원리에 1:1 매핑된다:

| 원리 | 설명 | 적용된 prop |
|---|---|---|
| **Stag Hunt** | 큰 보상은 commitment(시간/위치 고정)가 필요. 도중에 포기하면 손실. | 사당 기도 / 비석 파괴 / 저주 토템 |
| **Risk-Reward Asymmetry** | 위험이 클수록 보상이 비대칭적으로 커짐. 작은 commitment 는 작은 보상. | 사당 / 저주 토템 / 압전판 |
| **Tragedy of Commons** | 공유 자원을 욕심내면 모두가 손해. 절제가 최적. | 등불 (적 점거 시 dark) |
| **Chicken Game** | 상대를 유인하고 본인은 빠지기. 누가 먼저 무너지느냐. | 압전판 (적 유인 → 본인 회피) |
| **Information Asymmetry** | 플레이어 본인의 상태 정보가 비대칭적 정보 우위로 작용. | 별먼지 (HP 상태별 적응형 보상) |
| **Spatial Control** | 위치 자체가 자원. 누가 어느 라인을 점유하느냐. | 거울 파편 (발사체 반사) |
| **Coordination** | 부분/완전 합의(commitment)에 따라 다른 보상. | 잔해 점진 채굴 (1/2/3 단계) |

| 바위 ◇ | Spatial Cover | HP 60 / 0HP=80px 흙폭발+코인×3 / 적 발사체 차단 | shooter 적 라인 엄폐 |
| 유적 ✦ | Loss for Power | HP 100 / 0HP=5s atk+40% / spd+20% | 엄폐 잃고 화력 spike |
| 거울 파편 ◈ | Spatial Mind Game | 솔리드 + 모든 발사체 반사 (적 발사체 시안색 변환 후 적에게 데미지) | 적과 거울 사이 라인 점유 |

> 합계: 보상형 6 + 위험형 3 + 솔리드형 4 = **13종**. 모든 prop 의 의사결정 로직은 18개 회귀 테스트로 1:1 검증한다.

### 5-3. 의사결정 깊이 예시

**상황**: W6, 라이프 1/3, 보스 직전. 시야에 사당 / 잔해 / 압전판 + 적 무리.

**일반 서바이벌 게임에선**: 사당으로 달려가 깬다 (코인 획득). 끝.

**SAMSARA 에선**:
1. **사당 Pray** — 3초 정지 = 영구 +1 maxHP (보스 직전 결정타). 단, 정지 = 적 무리에 노출.
2. **잔해 점진** — 1히트만 = heart 즉시 (라이프 회복). 보스 후 다시 와서 폭발 가능.
3. **압전판 유인** — 적 무리를 압전판 위로 끌고 → 0.8s 안에 본인 빠짐 → 220px nova 로 절멸. 단, 미스 시 본인 -1 HP = 즉사.

3개 선택지가 **각각 다른 게임이론 원리** 에 따라 다른 시나리오에서 최적. 빌드와 직교하는 의사결정 축이 추가된다.

### 5-4. 평가 항목 기여도 (자체 분석)

| 평가 항목 (배점) | 13 prop 게임이론 기여 |
|---|---|
| **참신성 (20)** | 클리커 + 로그라이트에 **게임이론 기반 맵 결정**은 전례 거의 없음. 다른 출품작 차별화. |

## 6. 차별화 포인트

### 6-1. 트위스트 한 줄

> **"클리커의 즉각성 × 로그라이트의 빌드 다양성 × 30초 강제 사이클의 도파민 펄스 × 게임이론 기반 13 prop 결정 트리"**

#### 두 장르의 단점을 잘라낸 결과

- 기존 클리커 — 무한 진행으로 지루함.
- 기존 로그라이트 — 한 판 30분~1시간으로 길다.
- **SAMSARA** — 30초 사이클로 두 장르의 장점만 추출. 거기에 맵 결정의 깊이까지 더해 같은 카드 빌드라도 매번 다른 플레이 라인이 만들어진다.

#### 캐치프레이즈와 정체성의 같은 뿌리

본 트위스트는 §1-2.6 정체성 진술 5원칙의 *외부 노출 형태*다.

- P1 — 30초가 신성하다
- P2 — 모든 결정은 trade-off
- P3 — 죽음은 진보
- P4 — 읽지 않아도 알 수 있다
- P5 — 검증 가능

캐치프레이즈와 정체성이 같은 뿌리에서 나왔기 때문에, 마케팅 카피 *"30초마다 새 운명을 짠다"* 가 실제 게임 경험과 1:1 일치한다.
