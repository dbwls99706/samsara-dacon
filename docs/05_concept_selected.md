## 2. 코어 페이즈 — 3 단계 도파민 곡선

```
PHASE 1 — 발아 (웨이브 1-3, 약 90초)
  숫자 스케일: 1~100
  목표: 학습 완료, 첫 시너지 발현
  BGM: 베이스 라인만
  웨이브 3 끝: 첫 "어 이거 재미있네" 트리거

PHASE 2 — 개화 (웨이브 4-8, 약 150초)
  숫자 스케일: 100~5만
  목표: Run Identity 발현, 첫 보스 격파
  BGM: 베이스 + 드럼
  웨이브 5: 첫 보스 (목표 점수 30초 안에 격파)
  웨이브 8: 첫 시너지 5장 도달 → 메이저 능력 발동

PHASE 3 — 승천 (웨이브 9+, 죽을 때까지)
  숫자 스케일: 100만 → 10억 → 1조 → 1.4e15
  목표: 카오스 폭주, "헐" 순간 매초
  BGM: 4 레이어 풀 가동
  웨이브 10: 메가 보스
  웨이브 25: 신성 보스 (전설 카드 직접 선택)
  웨이브 50+: 점수 1e15 = 초월 엔딩 발동 가능
```

## 3. 핵심 루프 (사이클 1회 = 30초~35초)

```
[웨이브 시작]
   ├── 시작 모디파이어 1개 무작위 적용 (30종)
   └── BGM 진입
   ↓
[30초 탭 페이즈]
   ├── 탭 → 코인 + 콤보 + 효과
   ├── 자동/패시브 카드 작동
   ├── 트리거 카드 작동 (n번째 탭, 시간 경과 등)
   ├── 시너지 발동 (실시간)
   └── 핫스팟 등장 (1초마다 황금 영역)
   ↓
[웨이브 종료, 1초 정지 + 점수 카운트업]
   ↓
[카드 선택 — 3장 중 1장]
   ├── 보스 웨이브면: 의식 화면 (영구 런 버프)
   ├── 잭팟 1% 확률로 전설 카드 등장
   └── 듀얼 태그 카드 가능
   ↓
[다음 웨이브] → 반복
   ↓
(라이프 0 = 게임 오버)
   ↓
[게임 오버 화면]
   ├── 점수 + Run Identity 배너
   ├── 메타 RP(환생 포인트) 환산
   ├── "한 끗 부족" 표기
   ├── 하이라이트 릴 5초 자동 편집
   ├── 공유 이미지 생성
   └── [한 판 더] (1탭, 1.5초 안에 시작)
```

## 9. 메타 진보 (윤회 시스템)

- **환생 포인트(RP)**: 누적 코인 1만 = 1 RP
- RP로 잠금 해제:
  - 카드 1장 (RP 5) — 첫 30장 잠금, 점진 해제
  - 모디파이어 1종 (RP 8)
  - 캐릭터 스킨 1종 (RP 30) — 호랑이/까치/용/구미호/도깨비
  - BGM 트랙 (RP 50)
  - 시작 라이프 +1 (RP 100, 3회까지)
  - 시작 카드 +1 (RP 200, 1회)

| 누적 사이클 | 메인 화면 변화 |

## 10. 발견 / 비밀 / 업적

### 비밀 카드 5장
- "재의 봉황": 콤보 100 도달 + 불 카드 0장 클리어 시 풀에 추가
- "심해": 한 웨이브에서 탭 0회로 클리어 (자동만으로) 시
- "역설": 시간 카드 5장 + 한 웨이브 시간 0초로 클리어
- "광휘": 모든 6태그 1장씩 보유한 상태로 보스 격파
- "공허": 카드 0장으로 웨이브 5 도달

### 비밀 모디파이어 3종
- "최후의 주사위": 마지막 라이프에서 점수 ×10
- "역린": 콤보가 끊기면 그 시점까지 점수 ×3
- "거울 세계": 화면 좌우 반전

### 비밀 엔딩
- **초월 엔딩**: 점수 1e15 도달 → 5초 풀스크린 시네마틱 + 영구 트로피 + 메인 화면 변화 + 공유 가능 인증 이미지

### 업적 50개
- 단계 도전 (총 코인, 사이클, 콤보)
- 시너지 컬렉션 (각 18 시너지 첫 발동)
- 카드 컬렉션 (60장 모두 한 번 픽)
- 빌드 도전 (모든 Run Identity 11종 달성)
- 일일 도전 (연속 7일/30일 플레이)

## 11. 액티브 스킬 표현 (실력 = 점수)

클리커지만 실력이 점수에 명확히 반영:

1. **콤보 리듬** — 0.3초 윈도우. 일정 리듬 유지 시 콤보 ↑
2. **핫스팟** — 1초마다 화면에 1초 머무는 황금 영역. 그 위 탭 = ×3
3. **퀵타임 이벤트** — 콤보 ×25 도달 시 화면에 4 방향 패턴 → 정확히 따라 탭하면 ×100
4. **카드 우선순위** — 어떤 시너지 노릴지가 빌드 결정
5. **보스 격파 최적화** — 30초 내 목표 격파 시간 단축 → 보너스
6. **모디파이어 활용** — 같은 모디파이어를 어떻게 쓰는지가 점수 차이
